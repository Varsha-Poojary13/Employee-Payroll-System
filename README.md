# Employee Payroll System (C)

## Description

This project is a simple Employee Payroll System implemented in C using structures.
It allows users to add employee details and calculate their net salary.

## Features

* Add Employee Details
* Calculate Net Salary
* Display Employee Payroll
* Menu Driven Program
* Uses Structures in C

## Salary Formula

Net Salary = Basic Pay + Allowances - Deductions

## Technologies Used

* C Programming
* Structures
* Arrays
* Functions

## How to Run

Compile the program:

gcc payroll.c -o payroll

Run the program:

./payroll

## Example

Employee details are entered by the user and the system calculates the net salary automatically.
